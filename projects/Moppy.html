Project Overview

Moppy is a computer vision project focused on object detection for personal items, specifically wallets. This is the first phase of what will eventually become an autonomous "Find My Stuff" robot. Currently, the project implements a custom YOLOv11 model for wallet detection with professional MLOps practices.

üí° What's Been Built

A complete MLOps pipeline for custom object detection:

  • Custom Dataset: Curated and labeled dataset of wallet images for training
  • YOLOv11 Model: Trained specifically for wallet detection with high accuracy
  • NCNN Optimization: Model converted to NCNN format for efficient edge deployment
  • MLOps Integration: Complete workflow with MLflow for experiment tracking and DVC for data versioning
  • Data Management: DVC (Data Version Control) manages the custom wallet dataset, ensuring version control for both data and models while enabling reproducible experiments across different development environments.
  • Experiment Tracking: MLflow provides comprehensive experiment tracking, logging training metrics, hyperparameters, model artifacts, and performance comparisons to optimize model development.
  • Model Optimization: The trained YOLOv11 PyTorch model undergoes conversion to NCNN format, optimizing it for efficient inference on edge devices while maintaining detection accuracy.
  • Automated Pipeline: Streamlined workflow from data preparation through model training, evaluation, and deployment-ready model conversion, ensuring consistent and reliable model development.

Technologies Used

  • Machine Learning: YOLOv11, PyTorch, Custom Object Detection
  • MLOps: MLflow (experiment tracking), DVC (data versioning)
  • Edge Computing: NCNN Inference Engine for optimized deployment
  • Computer Vision: Custom dataset curation and model training
  • Development: Python, automated ML pipeline
